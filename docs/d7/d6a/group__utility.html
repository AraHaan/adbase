使用方法
Note
请勿直接引用对应头文件，util 模块统一引用<adbase/Utility.hpp> 文件
Example
#include <adbase/Utility.hpp>
.....
const string input = "dGhlIHNhbXBsZSBub25jZQ==258EAFA5-E914-47DA-95CA-C5AB0DC85B11";

checksum.update(input);
char hash[20] = {0};
checksum.final(hash);
